# **App Name**: Vigilant Exam

## Core Features:

- Detect Movement: Head Movement Detection: Analyze video feed to detect head movements.
- AI flag: Flag Cheating: Flag instances when excessive or suspicious head movements are detected.
- Live video: Live Video Display: Show live video stream for monitoring.
- notification: Flag Notifications: Display real-time notifications when potential cheating is detected. The LLM tool will reason about whether the candidate is acting suspicious and trigger a flag accordingly.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) to convey seriousness and professionalism.
- Background color: Light gray (#EEEEEE) to provide a neutral, unobtrusive backdrop.
- Accent color: Red-orange (#FF5722) for alerts and critical notifications to immediately draw attention.
- Body and headline font: 'Inter' (sans-serif) for clear and modern readability.
- Use clear, recognizable icons to represent alerts and monitoring status.
- Clean and simple layout with a focus on the live video feed. Notifications should be prominent but not obstructive.
